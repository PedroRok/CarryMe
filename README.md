### Welcome to Carry Me!

**Carry Me** is a lightweight addon for the **Carry On** mod that adds an explicit consent system for being carried.

Players can decide with a simple command or keybind whether they allow themselves to be picked up.  
Server owners can also enforce this behavior globally using a gamerule, making it ideal for multiplayer environments.
